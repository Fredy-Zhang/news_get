<p>The Australian government has triggered a <a href="https://www.abc.net.au/news/2020-04-28/government-calls-chinese-ambassador-boycott-coronavirus-inquiry/12191984" data-versionurl="http://web.archive.org/web/20200502113357/https://www.abc.net.au/news/2020-04-28/government-calls-chinese-ambassador-boycott-coronavirus-inquiry/12191984" data-versiondate="2020-05-02T11:33:59+00:00" data-amber-behavior="" target="_blank" rel="noopener">major clash</a> with Beijing by questioning the origin of COVID-19 &#8212; Prime Minister Scott Morrison and some of his senior cabinet members have publicly called for an independent inquiry into the coronavirus.</p>
<p>Although it has been seen by some as a thinly disguised attack on China, there has been considerable support down under. Morrison has <a href="https://www.theguardian.com/world/2020/apr/29/australia-defends-plan-to-investigate-china-over-covid-19-outbreak-as-row-deepens" data-versionurl="http://web.archive.org/web/20200504105346/https://www.theguardian.com/world/2020/apr/29/australia-defends-plan-to-investigate-china-over-covid-19-outbreak-as-row-deepens" data-versiondate="2020-05-04T10:53:47+00:00" data-amber-behavior="" target="_blank" rel="noopener">called on Australia&#39;s allies</a> to reform the World Health Organization and conduct an investigation of the coronavirus source.</p>
<p>Australia already faced a <a href="https://globalvoices.org/2020/02/18/china-travel-ban-heightens-threats-to-australias-trade-tourism-and-education-sectors/" target="_blank" rel="noopener">bleak future</a> in its economic relations with China as a result of the COVID-19 crisis. The shutdown of travel in early March was an early cause of strain between the countries. The lack of Australian government support for temporary visa holders such as <a href="https://www.theguardian.com/australia-news/2020/apr/17/we-feel-abandoned-international-students-in-australia-facing-coronavirus-alone" data-versionurl="http://web.archive.org/web/20200504110055/https://www.theguardian.com/australia-news/2020/apr/17/we-feel-abandoned-international-students-in-australia-facing-coronavirus-alone" data-versiondate="2020-05-04T11:00:57+00:00" data-amber-behavior="" target="_blank" rel="noopener">international students</a> and <a href="https://theconversation.com/90-000-foreign-graduates-are-stuck-in-australia-without-financial-support-its-a-humanitarian-and-economic-crisis-in-the-making-136717" data-versionurl="http://web.archive.org/web/20200504103854/https://theconversation.com/90-000-foreign-graduates-are-stuck-in-australia-without-financial-support-its-a-humanitarian-and-economic-crisis-in-the-making-136717" data-versiondate="2020-05-04T10:38:55+00:00" data-amber-behavior="" target="_blank" rel="noopener">graduates</a> has threatened the university sector.</p>
<p>The Chinese ambassador Cheng Jingye’s <a href="https://www.smh.com.au/politics/federal/australia-could-lose-billions-from-chinese-government-boycott-threat-20200427-p54nmh.html" data-versionurl="http://web.archive.org/web/20200504103756/https://www.smh.com.au/politics/federal/australia-could-lose-billions-from-chinese-government-boycott-threat-20200427-p54nmh.html" data-versiondate="2020-05-04T10:37:57+00:00" data-amber-behavior="" target="_blank" rel="noopener">response</a> was swift. He suggested that Chinese consumers and international students boycott Australian goods and services &#8212; especially the trade, tourism and education sectors, which have been vital to Australia&#39;s prosperity in recent decades.</p>
<p>A political and diplomatic brawl followed, with the Chinese embassy releasing alleged <a href="https://www.abc.net.au/news/2020-04-28/government-calls-chinese-ambassador-boycott-coronavirus-inquiry/12191984" data-versionurl="http://web.archive.org/web/20200502113357/https://www.abc.net.au/news/2020-04-28/government-calls-chinese-ambassador-boycott-coronavirus-inquiry/12191984" data-versiondate="2020-05-02T11:33:59+00:00" data-amber-behavior="" target="_blank" rel="noopener">details of a conversation</a> with Frances Adamson, the head of Australia’s Foreign Affairs and Trade department (DFAT). The release was contentious for two reasons. It was meant to show that the Australian government was backing off. It also breached customary diplomatic protocols.</p>
<p>The Chinese embassy claimed that:</p>
<blockquote><p>She also admitted it is not the time to commence the review now and Australia has no details of the proposal. She further said that Australia does not want the matter to have any impact on the Australia-China relationship.</p></blockquote>
<p>DFAT neither confirmed nor denied the claim:</p>
<blockquote><p>The department will not respond by itself breaching the long standing diplomatic courtesies and professional practices to which it will continue to adhere.</p></blockquote>
<p>This tweet was a typical reaction:</p>
<blockquote class="twitter-tweet">

<p>The editor of Chinese state media Global Times, Hu Xijin, added <a href="https://www.sbs.com.au/news/china-labels-australia-gum-stuck-to-the-bottom-of-china-s-shoe-over-calls-for-coronavirus-inquiry" data-versionurl="http://web.archive.org/web/20200504104258/https://www.sbs.com.au/news/china-labels-australia-gum-stuck-to-the-bottom-of-china-s-shoe-over-calls-for-coronavirus-inquiry" data-versiondate="2020-05-04T10:42:59+00:00" data-amber-behavior="" target="_blank" rel="noopener">fuel</a> to the controversy on Chinese social media platform Weibo:</p>

<blockquote><p>…it affords the Chinese what Australian policymakers should recognise as an unacceptable level of leverage, even a stranglehold, in times of stress.</p>
<p>…It is hard to escape the conclusion that Australia’s China policy is in the hands of amateurs, or ideologues, or both.</p></blockquote>

				return true;
			}

<script type="text/javascript">
	jQuery(document).ready(function($) {
		/**
		 * Fixed position sharing tools 
		 * 
		 * Makes sharing tools fixed position only when the viewport is mid-post
		 * 
		 * Loosely based on http://jqueryfordesigners.com/fixed-floating-elements/
		 */
		
		/**
		 * Get the jQuery objects to operate on
		 * 
		 * Run this ONLY ONCE to avoid re-quiering DOM on every scroll+resize event. 
		 */
		// Parent of fixed element, should be absolute positioned.
		var fixedParent = $('.post-header-sharing');
		// Fixed element, should have top:100px; to match fourth argument of update_fixed_element below
		var fixedElement = $('.sharing-tools-floating');
		// Container to match bottom to, must be position:relative
		var container = $('#single-post');

		/**
		 * Update the fixed element for the initial pageload
		 */
		update_fixed_element(fixedParent, fixedElement, container, 100);
		
		/**
		 * When the window is resized, updated the fixed status
		 */
		$(window).resize(function () {			
			update_fixed_element(fixedParent, fixedElement, container, 100);
		});

		// performance-enhancing timer http://ejohn.org/blog/learning-from-twitter/#comment-391945
//		var fixedElementScrollTimer = 0;

		/**
		 * When window is scrolled update the fixed status
		 */
		$(window).scroll(function () {
		
			// Use a buffer so we don't call myCallback too often.
//			if (fixedElementScrollTimer) {
//				clearTimeout(fixedElementScrollTimer);
//			}
//			fixedElementScrollTimer = setTimeout(myCallback, 100);

			update_fixed_element(fixedParent, fixedElement, container, 100);
		});

		/**
		 * Update the position:fixed status of fixedElement inside fixedParent
		 * 
		 * Checks the position of the parent and element in relation to the viewport and
		 * makes fixedElement position:fixed; if it would otherwise have scrolled out
		 * of view.
		 * 
		 * Also disables the fixed position if the element is at the bottom of container, instead
		 * making fixedParent absolute positioned at the bottom of container. 
		 * 
		 * @param object fixedParent jQuery object with results of search for the container div for fixedElement (must be absolute positioned)
		 * @param object fixedElement jQuery object with results of search for the element that will sometimes be position:fixed
		 * @param integer topSpacing Vertical height in pixes of "padding" above fixed element when position:fixed;
		 */
		function update_fixed_element(fixedParent, fixedElement, container, topSpacing) {
			
			// get the distance the viewport has scrolled down the page 
			var scrollPosition = $(window).scrollTop();
	
			// get the distance from the top of screen to the top of the parent element
			fixedParentTop = fixedParent.offset().top;
			
			// Get the bottom position fixed element itself
			fixedElementBottom = fixedElement.offset().top + fixedElement.height();
			
			// Get the bottom position of the container
			containerBottom = container.offset().top + container.height();
						
			// Get the minimum scrollPosition at which we'll make fixedElement fixed
			// i.e. its natural position minus our padding/spacing so that we start early
			fixedScrollPosition = container.offset().top - topSpacing;
			
			// Get the minimum scrollPosition at which we'll stick fixedParent to the bottom of container
			// i.e. the point when fixedElement would be outside container
			bottomScrollPosition = containerBottom - fixedElement.height() - topSpacing;
			
//			console.log('fixedElementBottom:' + fixedElementBottom + ' containerBottom:' + containerBottom + ' fixedScrollPosition:' + fixedScrollPosition + ' scrollPosition:' + scrollPosition + '');
//			console.log('fixedElement.height:' + fixedElement.height());
//			console.log('fixedParent.height:' + fixedParent.height());
//			console.log('fixedParentTop:' + fixedParentTop);
//			console.log('fixedScrollPosition:' + fixedScrollPosition);
//			console.log('bottomScrollPosition:' + bottomScrollPosition);
//			console.log('fixedElementBottom:' + fixedElementBottom);
//			console.log('containerBottom:' + containerBottom);

			 * If the window scroll is less than the fixed starting position, set it to default positioning
			 */
			if (scrollPosition < fixedScrollPosition) {
				// Make it static (default)
				fixedElement.css('position', 'static');
				// Reset the parent's bottom in case it was set to zero
				fixedParent.css('bottom', 'auto');

			/**
			 * If we have scrolled passed the bottom position (where fixedElement hits bottom of container) stick fixedParent to bottom
			 */
			}  else if (scrollPosition >= bottomScrollPosition) {
				// Set the element itself to static
				fixedElement.css('position', 'static');
				// Set the absolute-positioned parent to be at the bottom
				fixedParent.css('bottom', '0');

				//console.log ('fixedElementBottom (' + fixedElementBottom + ') >= containerBottom (' + containerBottom + ') -- make it bottom');
			
			/**
			 * Otherwise make fixedElement position:fixed so it can float up and down inside container
			 */
			} else {
				fixedElement.css('position', 'fixed');
				// Reset the absolute-positioned parent's bottom case it was set to 0
				fixedParent.css('bottom', 'auto');

				//console.log ('fixedElementBottom (' + fixedElementBottom + ') < containerBottom (' + containerBottom + ') -- make it fixed');
			};

		function gv_toggle_font_icon_arrow(parent_selector) {
			// Define the font icon classes to add to span

			// Get the .icon and make sure it exists

				arrow_icon.addClass(icon_down_class);
			} else if (arrow_icon.hasClass(icon_down_class)) {
				arrow_icon.removeClass(icon_down_class);
				arrow_icon.addClass(icon_up_class);
			}
		}

	function gv_data_close() {
		var cookies = $.cookie();

		// Scan all data-close elements
		$('[data-close]').each(function(){
			var id = $(this).attr('data-close');

			// Hide the element if gv_hide_$id cookie is present
			if (typeof $.cookie('gv_hide_' + id) !== 'undefined') {
				$('#' + id).hide();
			// Explicitly show the element if the cookie doesn't exist
			} else if (typeof $.cookie('gv_hide_' + id) === 'undefined') {
				$('#' + id).show();
			}
		});

		// Add on click events to all data-close elements
		$('[data-close]').on('click', function() {
			var id = $(this).attr('data-close');

			$.cookie('gv_hide_' + id, true, { path: '/', expires: 30 });
			$('#' + id).hide();
		});

	function gv_wpp_popup_js_fixes() {
		var cookies = $.cookie();
		var wpp_popup = $('[id^=wpp_html_theme-auto_popup-]');
		var wpp_popup_active = false;
		var wpp_popup_cookie_name = null;

		// Checking the length of an element is used to determined if it exists
		if (wpp_popup.length) {
			var match = /^wpp_html_theme-auto_popup-(.+)/im.exec(wpp_popup.attr('id'));

			// Extra validation to make sure we found a match.
			if (null !== match) {
				wpp_popup_cookie_name = 'wpp_popup_' + match[1] + '_closed'
				wpp_popup_active = true;
			}
		}

		// Scan all cookies for WP Popup cookie to see if it is closed. Update flag if found.
		for (var name in cookies) {
			if (wpp_popup_cookie_name === name) {
				wpp_popup_active = false;
				break;
			}
		}

		// Don't hide anything if popup isn't active.
		if (wpp_popup_active === false) {
			return;
		}

		// Scan all data-close elements and close the ID they refer to
		$('[data-close]').each(function(){
			var id = $(this).attr('data-close');
			$('#' + id).hide();
		});
	}
