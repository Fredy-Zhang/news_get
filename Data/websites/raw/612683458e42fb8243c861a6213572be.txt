                                                                <p><span lang="EN-US">Research by Polish scientists has revealed that billions of lightyears away the laws of physics are different from the ones on Earth.</span></p>
                                <p>The laws of physics describe or predict natural phenomena, such as what happens when someone drops an object out the window. Whether the window is in Poland or Australia, we know that the object will fall to the ground.</p><p>But new research by an international team of astrophysicists, including ones from Poland, indicates that the laws of physics are not the same everywhere. The findings come from the analysis of a signal from a quasar – an extremely luminous active galactic nucleus – from 13 billion years ago.</p><p><span class="fr-img-caption fr-fic fr-dib" style="width: 300px; width: 300px;"><span class="fr-img-wrap"><img src="https://thefirstnews-cms.s3.eu-central-1.amazonaws.com/jtj4465jwwwrwatglgdsd.jpeg" data-source="" data-author="Mariusz P. Dąbrowski" data-id="13894" data-type="img" data-name="Professor Mariusz P. Dabrowski's research confirms that changes in the laws of physics are not uniform throughout the universe."><span class="fr-inner fr-caption">Professor Mariusz P. Dąbrowski's research confirms that changes in the laws of physics are not uniform throughout the universe.<span class="fr-text-author">Mariusz P. Dąbrowski</span></span></span></span></p><p>The team presented its findings in an article entitled ‘Four direct measurements of the fine-structure constant 13 billion years ago’ published in the scientific journal ‘Science Advances’ on April 24, which was co-written by scientists from the University of Szczecin and the Maritime University in Szczecin along with other research institutions in Europe, Australia and the United States.</p><p>“In our research, we think about whether the laws of physics here and now are the same as somewhere else long ago. And it turns out that they are not,” said Professor Mariusz P. Dąbrowski from the University of Szczecin, one of the authors of the article.</p><p>The scientists studied the quasar J1120+0641, which was formed a billion years after the Big Bang and is one of the most luminous quasars known. They used spectra obtained using the X-SHOOTER spectrograph on the European Southern Observatory’s Very Large Telescope (VLT), which is located in Chile.</p><p><span class="fr-img-caption fr-fic fr-dib" style="width: 300px; width: 300px;"><span class="fr-img-wrap"><img src="https://thefirstnews-cms.s3.eu-central-1.amazonaws.com/2o5l2q1d6y691kvqgx5tho.jpeg" data-source="" data-author="ESO/M. Kornmesser / CC BY (https://creativecommons.org/licenses/by/4.0)" data-id="13893" data-type="img" data-name="The quasar, J1120+0641, is 13 billion lightyears away."><span class="fr-inner fr-caption">The quasar, J1120+0641, is 13 billion lightyears away.<span class="fr-text-author">ESO/M. Kornmesser / CC BY (https://creativecommons.org/licenses/by/4.0)</span></span></span></span></p><p>“We found that a certain physical constant – so-called fine-structure constant (alpha) – may have been smaller in that period than it is now,” explained Dąbrowski, adding that the constant determines how electric charges attract.</p><p>More generally, the research confirms that changes in the laws of physics are not uniform throughout the universe, which has implications for scientists’ understanding of symmetry in the cosmos.</p><p>“If some points are at the same distance from us – for example, 13 billion light years away – but we look at one in the northern direction and the other in the western direction, it could turn out that the value of the fine-structure constant will be different there. And this means that the change in this constant in the universe is heterogeneous,” said Dąbrowski.</p>

        var pageTop = $(window).scrollTop();
        var pageBottom = pageTop + $(window).height();
        var elementTop = $(element).offset().top;
        var elementBottom = elementTop + $(element).height();
        return ((elementTop <= pageBottom) && (elementBottom >= pageTop));
    }

    function removeElementsByAtrrAndID(elements, attrName, id) {
        return elements.not("[" + attrName + "=" + id + "]");
    }

    function removeIdFromSessionStorage(item, storageName) {
        var index = getSessionStorageData(storageName).indexOf(item);
        if (index > -1) {
            var sessionDataWithoutRemovedItem = getSessionStorageData(storageName).splice(index, 1);
            setSessionStorage(sessionDataWithoutRemovedItem, storageName);
        }
    }

    function addDataToSessionStorage(data, storageName) {
        data.forEach(function (item) {
            addIdToSessionStorage(item, storageName);
        })
    }

    function addIdToSessionStorage(id, storageName) {
        var storageIds = getSessionStorageData(storageName);

        if (!storageIds) {
            storageIds = [];
        }

        if (!storageIds.includes(id)) {
            storageIds.push(id);
        }
        setSessionStorage(storageIds, storageName);
    }

    function isIdInSessionStorage(id, storageName) {
        var storage = JSON.parse(sessionStorage.getItem(storageName));
        if (storage) {
            return storage.includes(id);
        } else return false;
    }

    function setSessionStorage(idArray, storageName) {
        sessionStorage.setItem(storageName, JSON.stringify(idArray));
    }

    function getSessionStorageData(storageName) {
        var storageData = JSON.parse(sessionStorage.getItem(storageName));
        if (!storageData) {
            return [];
        } else return storageData;
    }

    function getElementIdByAttrName(element, attrName) {
        return $(element).attr(attrName);
    }

    function removeElementByAttrIfIdInStorage(elements, attrName, storageName) {
        var elementDataId;
        elements.each(function () {
            elementDataId = getElementIdByAttrName(this, attrName);
            if (isIdInSessionStorage(elementDataId, storageName)) {
                elements = removeElementsByAtrrAndID(elements, attrName, elementDataId);
            }
        });

        return elements;
    }

                    viewIds.push(elementDataId);

                }
            }
        });

        return viewIds;
    }

    function convert(img){
        var img = $(img);
        var avalibleHeight = [
            50, 100, 150, 200, 250, 260, 285, 295, 300, 350, 380, 400, 500, 600, 610, 700, 800, 820, 1000, 1240, 1400, 1800, 2200, 2600, 3000
        ];
        var width = Math.round(img.width());
        width = closest(avalibleHeight, width);

        if(typeof ratio === 'undefined')
        {
            ratio = 1.5;
        }

        var baseUrl = img.data('src');
        if(typeof baseUrl === 'undefined')

            baseUrl = img.attr('src');
        }

        var s3url = 'https://thefirstnews-cms.s3.eu-central-1.amazonaws.com';

        var urlsToCheck = [
            s3url,
            s3urlAlias,
            lambdaUrl
        ];

        var newS3Url = baseUrl;
        for(var i in urlsToCheck) {

            }

            img.height(95);
        }))
    }

    function closest(array,num){
        var ans;
        if(num <= array[0]){
            ans = num;
        }else if(num >= array[array.length-1]){
            ans = array[array.length-1];
        }else{
            var i=0;
            for(i in array){
                var next = parseInt(i)+1;
                if(array[i] <= num && num < array[next] )
                {
                    ans = array[next];
                }
            }
        }
        return ans;
    }
