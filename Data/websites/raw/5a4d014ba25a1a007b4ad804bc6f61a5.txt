function fetchtimeout(ms,url,options,fn) {

let didTimeOut = false;
return new Promise(function(resolve, reject) {
const timeout = setTimeout(function() {
didTimeOut = true;
reject(new Error('Request timed out -> ' +ms +" ms"));
}, FETCH_TIMEOUT);
fetch(url,options).then(function(response) {
// Clear the timeout as cleanup
clearTimeout(timeout);
if(!didTimeOut) {
resolve(response);
}
})
.catch(function(err) {
// Rejection already happened with setTimeout
if(didTimeOut) return;
// Reject with error
reject(err);
//	console.error(err.message +"= "+url);
})
}).then(function(response) {
// Request success and no timeout
if(response.ok) {
return response.json();
}
else
{
console.error('Network response was not ok.' + url + "-"+ response.statusText); }
})
.then(function(response) {
fn(response);
})
.catch(function(err) {
// Error: response error, request timeout or runtime error

})
}

<h2 class="intro">
Two militants were also killed during the 18-hour operation </h2>
<div id="content-body-14269002-31493302">
<p>Five security personnel, including a Colonel, a Major and a sub-inspector, and two unidentified militants were killed during an 18-hour anti-militancy operation in north Kashmir’s Handwara area.</p><p>“In a firefight in Changimul, Handwara of Kupwara district, two terrorists were eliminated and the team of five security forces personnel, comprising two Army officers, two soldiers and one police sub-inspector, also died,” a Srinagar-based Army spokesman said,</p><p>The spokesman said the operation was launched after the intelligence inputs were received that terrorists were taking civilian inmates of a house in Changimul village hostage.</p><p>“A joint operation was launched by the Army and the police on Saturday. A team comprising five Army and police personnel entered the target area occupied by the terrorists to evacuate the civilians. They successfully extricated the civilians. However, during the process, the team was subjected to heavy volume of fire by the terrorists,” said the Army.</p><p>Police officials said the operation was called off on Sunday morning after the bodies of the security personnel and the militants were spotted inside the house at the encounter site in Changimul area, nestled in a forest zone.</p><p>The deceased senior Army officers were identified as Colonel Ashutosh Sharma, Major Anuj Sood and Naik Rajesh and Dinesh, besides sub-Inspector Mohammad Sagier Qazi of J&amp;K Police’s Special Operation Group (SOG). One soldier was also injured in the operation, officials said.</p><p>Colonel Sharma, Commanding Officer of the Army’s 21 Rashtriya Rifles, was stationed in Handwara.</p><p>
