<div id="node-1062332" class="node node-news-story node-full clearfix" about="/en/story/2020/04/1062332" typeof="sioc:Item foaf:Document">

  <div class="content">
    <div class="field field-name-field-news-date field-type-datetime field-label-hidden"><div class="field-items"><div class="field-item even"><span class="date-display-single" property="dc:date" datatype="xsd:dateTime" content="2020-04-22T00:00:00-04:00">22 April 2020</span></div></div></div><div class="field field-name-field-news-topics field-type-entityreference field-label-hidden"><div class="field-items"><div class="field-item even"><a href="/en/news/topic/climate-change">Climate Change</a></div></div></div><div class="field field-name-field-news-story-lead field-type-text-long field-label-hidden"><div class="field-items"><div class="field-item even"><p>An expected drop in greenhouse gas emissions linked to the global economic crisis caused by the COVID-19 pandemic is only “short-term good news”, the head of the UN <a href="https://public.wmo.int/en" rel="noreferrer noopener" target="_blank">weather agency</a> said on Wednesday. </p>

  <div class="field field-name-field-news-story field-type-paragraphs field-label-hidden"><div class="field-items"><div class="field-item even"><div class="entity entity-paragraphs-item paragraphs-item-one-column-text" about="" typeof="">
  <div class="content">
    <div class="field field-name-field-text-column field-type-text-long field-label-hidden"><div class="field-items"><div class="field-item even"><p>“This drop of emissions of six per cent, that’s unfortunately (only) short-term good news”, said Professor Petteri Taalas, World Meteorological Organization (WMO) Secretary-General, in reference to a 5.5 to 5.7 per cent fall in levels of carbon dioxide due to the pandemic, that have been flagged by leading climate experts, including the Center for International Climate Research. </p>
<div class="dnd-widget-wrapper context-un_news_default type-twitter atom-align-right">
<div class="dnd-atom-rendered"><!-- scald=633942:un_news_default {"additionalClasses":""} --><blockquote class="twitter-tweet"><p lang="en" dir="ltr">In the 50 years since the first <a href="https://twitter.com/hashtag/EarthDay?src=hash&amp;ref_src=twsrc%5Etfw">#EarthDay</a>, <a href="https://twitter.com/hashtag/climatechange?src=hash&amp;ref_src=twsrc%5Etfw">#climatechange</a> has accelerated, reaching a new peak in the past 5 years, which were the hottest on record. <p>Once the global economy begins to recover from the new <a href="https://www.un.org/coronavirus" title="Coronavirus disease (COVID-19)" class="word-link" target="_blank">coronavirus</a>, WMO expects emissions to return to normal. </p>
<p>“There might even be a boost in emissions because some of the industries have been stopped”, the WMO head cautioned. </p>
<h3>CO2 at record levels once again </h3>
<p>Latest data from WMO published to coincide with the 50th anniversary of Earth Day, on 22 April, indicates that carbon dioxide (CO2) levels and other greenhouse gases in the atmosphere rose to new records last year. </p>
<p>Levels of carbon dioxide were 18 per cent higher from 2015 to 2019 than the previous five years, according to WMO’s <a href="https://library.wmo.int/index.php?lvl=notice_display&amp;id=21718#.Xp3JvVMzajR" rel="noreferrer noopener" target="_blank">Global Climate 2015-2019</a> report. </p>
<p>The gas “remains in the atmosphere and oceans for centuries. This means that the world is committed to continued climate change regardless of any temporary fall in emissions due to the <a href="https://www.un.org/coronavirus" title="Coronavirus disease (COVID-19)" class="word-link" target="_blank">Coronavirus</a> epidemic”, the report states, with data indicating that CO2 concentrations are on track “to reach or even exceed 410 ppm by the end of 2019”. </p>
<p>The forecasted fall in carbon emissions is mirrored by decreases in levels of common air pollutants from car exhausts and fossil fuel energy, such as nitrous oxide (N2O) particles. </p>
<p>“Their lifetime is typically from days to weeks, so the impact is seen more rapidly”, Professor Taalas said. “But these changes in the carbon emissions, they haven’t had any impact on climate so far.” </p>
<h3>Cleaner air even in most polluted cities</h3>
<p>Highlighting the dramatic improvement in air quality in major cities and industrialised regions “in several parts of the world”, the WMO chief noted that this has been the case “in China, in India and also here close to us in the Po Valley in northern Italy, which is one of the most polluted areas in Europe. And we have seen that also in individual cities like Paris.” </p>
<p>Climate change was “of a different magnitude” to the problems posed by the new coronavirus, Professor Taalas insisted, underscoring its serious health risk, along with a devastating economic impact. </p>
<h3>Mitigating climate change is a question of survival</h3>
<div class="dnd-widget-wrapper context-un_news_default type-image atom-align-left">
<div class="dnd-atom-rendered"><!-- scald=633952:un_news_default {"additionalClasses":""} --><img class="img-responsive" src="https://global.unitednations.entermediadb.net/assets/mediadb/services/module/asset/downloads/preset/assets/2018/05/2011-01-28_WMO_Flickr_Antartica.jpg/image560x340cropped.jpg" title="Ice floating on the waters of Prince Gustav Channel in Antarctica, where an ice shelf (Prince Gustav Ice Shelf) of more than 28 km used to exist.  The ice shelf has since retreated and collapsed. (file)" width="560px" height="315px" /><!-- END scald=633952 --></div>
</div>
<p>Given the fact that the last 50 years have seen the physical signs of climate change - and their impacts – gathering speed at a dangerous rate, the WMO Secretary-General insisted that unless the world can mitigate climate change, it will lead to “persistent health problems, especially hunger and inability to feed the growing population of the world and there would be also more massive impact on economics”. </p>
<p>Since the first Earth Day in 1970, carbon dioxide levels have gone up 26 per cent, and the world’s average temperature has increased by 0.86 degrees Celsius (33.5 Fahrenheit). </p>
<p>The planet is also 1.1C (nearly 34F) warmer than the pre-industrial era and this trend is expected to continue. </p>
<p>In its latest report warning of the impacts of climate change, the UN agency confirmed that the last five years were the hottest on record. </p>
<p>This warming has been uneven, with Europe seeing the highest change in the last decade (of around +0.5C, or 32.9F) and South America experiencing the least change. </p>
<p>Others key indicators showed an acceleration of climate change in the past five years. </p>
<p>These include ocean heat and acidification, rising sea level (up 112 millimetres since 1970), glacier melt and Arctic and Antarctic sea ice shedding (with ice loss five times higher in last five years, compared to the 1970s). </p>

var playListModel = function($, options) {

  // Private Variables
  var instance = this;
  var changed = true;
  var items = options.setup.playlist;
  var htmlPlaylist = document.getElementById(options.htmlListID);
  var playListItemTemplate = options.playListItemTemplate;
  var currentlyPlayingIndex = -1;

  var interactionStarted = false;

  // Setup the player instance
  var playerInstance = jwplayer(options.setup.id);
  playerInstance.setup(options.setup);

  // Event Handlers

  // OnReady
  playerInstance.on('ready',function() {
    // Update the playlist HTML with the internal items
    updatePlaylistHTML(instance.items);
  });

  // OnPlayListItem
  playerInstance.on('playlistItem',function() {
    // Update the current item. this = playerInstance
    var playlistItem = this.getPlaylistItem();
    if (parseInt(playlistItem.index) > -1) {
      instance.currentlyPlayingIndex = playlistItem.index;
    }
    // Send RadioActivity event, this is currently responsible for a concurrent editing bug.
    //updateViewCounter(playlistItem.nid);
    // Reflect the changes.
    updatePlaylistView();
  });

  playerInstance.on('complete',function() {

    // Update the current item. this = playerInstance
    var playlistItem = this.getPlaylistItem();
    if (parseInt(playlistItem.index) > -1) {
      instance.currentlyPlayingIndex = playlistItem.index;
    }

    // If the current item is 0
    // and the items list is longer than the players list
    // then start playing the next item. 
    if ((instance.currentlyPlayingIndex == 0) && (items.length > this.getPlaylist().length)) {
      instance.playThis(1);
      updatePlaylistView();
    }

  });

  // Private Functions

  updatePlaylistHTML = function(arrayList) {

    // Create the HTML items
    var output = '';
    for (var index = 0; index < arrayList.length; index++){
      output += createPlaylistItemHTML(index, arrayList[index]);
    }
    htmlPlaylist.innerHTML = output;

    // Set the active classes
    jQuery('li:nth('+instance.currentlyPlayingIndex+')', htmlPlaylist).addClass('active-pl-item');
    jQuery("li.active-pl-item .close-icon").hide();

    updatePlayer();
  }

  updatePlayer = function() {
    // Update Player metadata
    var playlistItem = playerInstance.getPlaylistItem();
    if (playlistItem.image != '') {
      jQuery( "div#unnews-playlist-image img" ).attr('src', playlistItem.image).addClass('');
      jQuery( "#unnews-player-meta" ).css('background-color', '#000000');
    }
  }

  createPlaylistItemHTML = function(index, item) {
    var playindex = index + 1;
    var shortTitle = shortenString(item.title, 60, true);
    var returnValue = playListItemTemplate;
    returnValue = returnValue.replace(/__index__/g, index);
    returnValue = returnValue.replace(/__item\.product__/g, item.product ? item.product : '');
    returnValue = returnValue.replace(/__item\.duration__/g, item.duration ? item.duration : '');
    returnValue = returnValue.replace(/__item\.file__/g, item.file);
    returnValue = returnValue.replace(/__shortTitle__/g, shortTitle);
    return returnValue;
  }

  shortenString = function(string, maxLength, addEllipsis) {

    var addEllipsis = addEllipsis || false;
    var maxLength = maxLength || 50;
    var string = jQuery.trim(string).substring(0, maxLength);

    if (addEllipsis && string.length >= maxLength) {
      string = string + "...";
    }

    return string;
  }

  isPlaying = function() {
    return (playerInstance.getState() === 'playing');
  }

  /* Update the inner playlist, html, and player as applicable. */
  updatePlaylistView = function() {

    // Only update the JWPlayer playlist
    // - if it is currently not playing
    // - if it needs to be updated (is changed)
    // else
    // - if it IS playing, set the changed bit to TRUE
    //   so we now to update the JWPlayer play list in
    //   the next possible window. (when not playing)
    if (!isPlaying() && instance.interactionStarted) {
      if (changed == true) {
        changed = false;

        playerInstance.pause();
        playerInstance.load(items);
        playerInstance.playlistItem(instance.currentlyPlayingIndex);
        // Exit out of the function, the event will
        // trigger the redraw.
      }
    } else {
      changed = true;
    }

    // Update the HTML
    updatePlaylistHTML(items);

  }

  debug = function() {
    console.log('state:' + playerInstance.getState());
    console.log('items', items);
    console.log('changed: '+ changed);
    console.log('currentlyPlayingIndex:' + instance.currentlyPlayingIndex);
    //console.log();
  }

  // Public Functions

  this.playThis = function(index) {
    // Reflect the changes.
    if ((currentlyPlayingIndex != index) || !isPlaying()) {
      updatePlaylistView();
      playerInstance.playlistItem(index);
    }
    instance.currentlyPlayingIndex = index;
  }

  // Add one or more items.
  this.addItems = function(arrayItemsValue, blnReset) {

    // Record change has happened...
    changed = true;

    // If resetting, stop the player.
    if (blnReset) {
      playerInstance.stop();
    }

    // If resetting, simply re-initialize the internal array.
    if (blnReset) {
      items = [];
    }

    // Check if we're adding multiple items.
    if (arrayItemsValue) {
      if ($.isArray(arrayItemsValue)) {
        for (var i = 0; i <= arrayItemsValue.length; i++) {
          if (arrayItemsValue[i] && (items.indexOf(arrayItemsValue[i]) == -1) ) {
            items.push(arrayItemsValue[i]);
          }
        }
      } else {
        items.push(arrayItemsValue);
      }
    }

    // Reflect the changes.
    updatePlaylistView();

    // If resetting, start playing after adding the items.
    if (blnReset) {
      playerInstance.play();
    }
  }

  this.removeItem = function(intIndex) {

    isStopped = !isPlaying();

    // Record change has happened...
    changed = true;

    // Get the item to be removed
    itemToBeRemoved = items[intIndex];

    // Remove the item by using a filter function.
    items = items.filter(function(itemToBeCompared){
       return itemToBeCompared != itemToBeRemoved;
    });

    // Reflect the changes.
    updatePlaylistView();

    if (isStopped){
        playerInstance.pause();
    }
  }

  this.debug = function() {
    console.log(this);
    return this;
  }

  // Properties
  this.items = function() {
    return items;
  }

  this.setInteractionStarted = function() {
    this.interactionStarted = true;
  }

  return instance;
};
