    function up() {
        var max_size = 48;
        var actualSize = $('#content .story-desc>p').css("font-size");
        if (parseInt(max_size) > parseInt(actualSize)) {
            var newSize = parseInt(actualSize) + 5;
            $('#content .story-desc>p').css({
                'font-size': newSize + "px"
            });
            UpdateCookies(newSize);
        }
    }

    function down() {
        var min_size = 18;
        var actualSize = $('#content .story-desc>p, #content .story-desc>p span').css("font-size");
        if (parseInt(min_size) < parseInt(actualSize)) {
            var newSize = parseInt(actualSize) - 5;
            $('#content .story-desc>p, #content .story-desc>p span').css({
                'font-size': newSize + "px"
            });
            UpdateCookies(newSize);
        }

    }

    function getCookieValue() {
        var a = 'fontcookie'
        var b = document.cookie.match('(^|[^;]+)\\s*' + a + '\\s*=\\s*([^;]+)');
        return b ? b.pop() : '';
    }

    function SetStorySize() {
        var cookie_string = '';
        var expiration_date = new Date();
        expiration_date.setFullYear(expiration_date.getFullYear() + 1);
        var cookieVal = getCookieValue();

        if (cookieVal == "") {
            cookieVal = "18px";
        }
        cookie_string = "fontcookie=" + cookieVal + "; path=/; expires=" + expiration_date.toUTCString();

        $('#content .story-desc>p, #content .story-desc>p span').css({
            'font-size': cookieVal
        });

        document.cookie = cookie_string;
    }

    function UpdateCookies(size) {
        size = size + "px";
        var expiration_date = new Date();
        expiration_date.setFullYear(expiration_date.getFullYear() + 1);
        cookie_string = "fontcookie=" + size + "; path=/; expires=" + expiration_date.toUTCString();
        document.cookie = cookie_string;
    }

                    <p><span style=";">New Delhi, May 4</span></p>
<p>The police, which on Monday initiated a probe against the Instagram chat group &ldquo;Boys Locker Room&rdquo;, have asked the photo sharing app to share the details of all participants in the group where photos of underage girls were being shared and objectified.</p>
<p>Anish Roy, Deputy Commissioner of Police (DCP), Cyber Crime, said, &ldquo;We have asked Instagram to share the details of the members and admin of the said group, including their names, IP addresses etc.&rdquo; Leaked screenshots of the private Instagram chat group kicked up a storm over culture in India after numerous boys were allegedly seen sharing photos of underage girls, objectifying them, and planning &lsquo;gang rapes&rsquo;.</p>
<p>Earlier in the day, the Cyber Crime Cell of the Delhi Police initiated a probe.</p>
<p>The incident came to light after a girl from south Delhi shared a screenshot on social media, exposing the group and scores of similar groups.</p>
<p>She wrote, &ldquo;A group of South Delhi guys aged 17-18 types have this IG group named &lsquo;Boy&rsquo;s locker room&rsquo; where objectify and morph pictures of girls their age. Two boys from my school are a part of it.&rdquo; The girl also shared the screenshot of the list of participants of the group and that of their chats where they could be seen sharing pictures of girls and passing comments over them.</p>
<p>The Delhi Commission for Women too swung into action and issued a notice to the Delhi Police and Instagram over the issue. &ldquo;I saw a screenshot of the &ldquo;Boys Locker Room&rdquo; Instagram group. This is a clear example of a gruesome criminal and rapist mindset,&rdquo; Delhi Commission for Women chief Swati Maliwal said.</p>
